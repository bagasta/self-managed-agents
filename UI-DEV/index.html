🚨 Escalation Simulator

Simulasi percakapan antara User dan Operator tanpa perlu WhatsApp API. Pesan dari operator dikirim via /v1/channels/incoming dengan from_phone = operator_phone.

⚙️ Setup
Cara pakai: 1️⃣ Pilih agent yang sudah punya escalation_config.operator_phone → 2️⃣ Isi Session ID → 3️⃣ Klik Force Aktifkan Eskalasi (atau biarkan agent memicunya sendiri lewat chat) → 4️⃣ Kirim pesan sebagai User di kiri, balas sebagai Operator di kanan
