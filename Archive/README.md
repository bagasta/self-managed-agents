# Archive

Folder ini menyimpan dokumen lama, artifact lokal, dan file eksperimen yang tidak lagi menjadi jalur utama project.

Isi archive dipisah menjadi:

- `docs/`: dokumentasi/planning lama yang masih berguna sebagai histori.
- `local-artifacts/`: file lokal, output test, virtualenv duplikat, log, sample media, atau prototype yang tidak perlu masuk git.

File aktif project tetap berada di lokasi utama seperti `README.md`, `CLAUDE.md`, `docs/recap.md`, `system-message-builder.md`, `app/`, `tests/`, `wa-service/`, dan `wa-dev-service/`.
